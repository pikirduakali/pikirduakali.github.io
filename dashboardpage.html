How to Import

  1. Compose — Isi title, tag, date, dan body artikel lo.
  2. Download .txt — Klik "Download .txt" buat dapet file artikel.
  3. Taruh file — Pindahin file .txt ke folder articles/ di project lo.
  4. Copy entry — Klik "Copy entry" buat copy baris JavaScript.
  5. Buka articles.html — Buka file articles.html di editor lo.
  6. Cari array articles — Cari bagian ini:
    const articles = [
      ...
      { tag: '...', ... },
    ];
  7. Paste — Tempel entry yang di-copy sebelum baris ];.
  8. Push — Commit & push ke GitHub. Selesai!
Apa itu array? Array itu daftar. Di articles.html, array articles (mulai dari const articles = [ sampai ];) adalah daftar semua artikel lo. Setiap baris di dalemnya adalah 1 artikel. Data entry yang lo copy dari dashboard tinggal lo tempelin di dalem kurung [ ], sebelum ];.
